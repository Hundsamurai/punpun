# PunPun - Анализатор вакансий для Obsidian

Плагин для анализа соответствия резюме вакансиям с помощью AI (DeepSeek).

## Возможности

- 📝 Управление базой вакансий
- 🤖 Анализ резюме через DeepSeek API
- 📊 Оценка соответствия и покрытия ключевых слов
- 💡 Рекомендации по улучшению резюме

## Установка

1. Скопируйте папку в `.obsidian/plugins/punpun`
2. Установите зависимости: `npm install`
3. Соберите плагин: `npm run build`
4. Включите плагин в настройках Obsidian

## Настройка

1. Откройте настройки плагина
2. Введите API ключ DeepSeek
3. Укажите путь к файлу с резюме (по умолчанию `resume.md`)
4. Укажите путь к базе вакансий (по умолчанию `vacancies.json`)

## Использование

### Добавление вакансии

Используйте команду "Добавить вакансию" через палитру команд (Ctrl/Cmd + P)

### Анализ резюме

Используйте команду "Анализировать резюме по всем вакансиям"

Плагин проанализирует ваше резюме по каждой вакансии и покажет:
- Процент соответствия
- Покрытие ключевых слов
- Отсутствующие навыки
- Рекомендации по улучшению

## Формат базы вакансий

Создайте файл `vacancies.json` в корне хранилища:

```json
[
  {
    "id": "1",
    "company": "Компания А",
    "title": "Frontend Developer",
    "url": "https://company.com/careers/frontend",
    "description": "Описание вакансии...",
    "keywords": ["React", "TypeScript", "CSS"]
  }
]
```

## Разработка

```bash
npm install
npm run dev
```
