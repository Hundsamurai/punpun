# PunPun - Анализатор вакансий для Obsidian

Плагин для анализа соответствия резюме вакансиям с помощью AI (DeepSeek или OpenRouter).

## Возможности

- 📝 Управление базой вакансий
- 🤖 Анализ резюме через DeepSeek API или OpenRouter
- 🔄 Поддержка множества моделей через OpenRouter (Claude, GPT-4, Gemini, Llama и др.)
- 📊 Оценка соответствия и покрытия ключевых слов
- 💡 Рекомендации по улучшению резюме
- ✉️ Генерация сопроводительных писем

## Установка

1. Скопируйте папку в `.obsidian/plugins/punpun`
2. Установите зависимости: `npm install`
3. Соберите плагин: `npm run build`
4. Включите плагин в настройках Obsidian

## Настройка

1. Откройте настройки плагина
2. Выберите AI провайдера:
   - **DeepSeek** - быстрый и экономичный вариант
   - **OpenRouter** - доступ к множеству моделей (Claude, GPT-4, Gemini и др.)
3. Введите соответствующий API ключ:
   - DeepSeek: получите на [platform.deepseek.com](https://platform.deepseek.com)
   - OpenRouter: получите на [openrouter.ai](https://openrouter.ai)
4. Для OpenRouter выберите модель из списка:
   - Claude 3.5 Sonnet (рекомендуется)
   - Claude 3 Opus
   - GPT-4 Turbo
   - GPT-4o
   - Gemini Pro 1.5
   - Llama 3.1 70B
   - 🆓 Nvidia Nemotron 120B (бесплатная)
   - 🆓 Arcee Trinity Large (бесплатная)
   - 🆓 GLM 4.5 Air (бесплатная)
   - 🆓 GPT OSS 120B (бесплатная)
5. Укажите путь к файлу с резюме (по умолчанию `resume.md`)
6. Укажите путь к базе вакансий (по умолчанию `vacancies.json`)

## Использование

### Добавление вакансии

Используйте команду "Добавить вакансию" через палитру команд (Ctrl/Cmd + P)

### Анализ резюме

- **Анализ всех вакансий**: "Анализировать резюме по всем вакансиям"
- **Анализ одной вакансии**: "Анализировать резюме по выбранной вакансии"

Плагин проанализирует ваше резюме и покажет:
- Процент соответствия (matchScore)
- Покрытие ключевых слов
- Детальную разбивку по категориям (hard skills, опыт, образование и т.д.)
- Отсутствующие и присутствующие навыки
- Рекомендации по улучшению
- Советы по оптимизации для ATS-систем
- Готовое сопроводительное письмо

## Формат базы вакансий

Создайте файл `vacancies.json` в корне хранилища:

```json
[
  {
    "id": "1",
    "company": "Компания А",
    "title": "Frontend Developer",
    "url": "https://company.com/careers/frontend",
    "description": "Описание вакансии...",
    "keywords": ["React", "TypeScript", "CSS"]
  }
]
```

## Сравнение провайдеров

### DeepSeek
- ✅ Быстрый и экономичный
- ✅ Хорошее качество анализа
- ✅ Низкая стоимость
- ⚠️ Одна модель

### OpenRouter
- ✅ Множество моделей на выбор
- ✅ Доступ к лучшим моделям (Claude, GPT-4)
- ✅ Бесплатные модели (Nemotron, Trinity, GLM, GPT OSS)
- ✅ Гибкость в выборе модели под задачу
- ⚠️ Платные модели могут быть дороже
